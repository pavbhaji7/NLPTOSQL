# 3.2.3 Architecture Document

The frontend architecture follows a modern Single Page Application (SPA) design, currently configured for static hosting (e.g., Vercel). 

The client layer consists of React components communicating via state hooks. When an interaction occurs in the `QueryInput`, an Axios/Fetch REST call is dispatched to an API Gateway. This Gateway securely routes the payload to the Cloud Function / Python server running the NLP pipeline.

Upon completion, the backend returns a complex JSON object. The frontend architecture specifically maps the `tokens` array from this payload to the `PipelineVisualizer` state, and maps the `final_sql` string to the `SQLResult` code-block component. This decoupled architecture ensures the UI remains highly responsive while the backend handles the heavy computational inference.
