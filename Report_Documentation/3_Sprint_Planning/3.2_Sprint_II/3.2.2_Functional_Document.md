# 3.2.2 Functional Document

During Sprint II, the frontend was built using React and styled with a sleek, vanilla CSS design system to ensure fast load times and a premium look.

The `SchemaViewer` component was implemented to dynamically render the JSON schema graph on the left panel, providing users with immediate context about tables and their relationships.

The `QueryInput` component handles asynchronous HTTP requests to the backend API Gateway.

The critical `PipelineVisualizer` component processes the JSON payload returned by the Python backend. It iterates over the tagged arrays and renders color-coded chips—for example, rendering 'Values' in green and 'Meta' schema elements in blue. This functional visualization guarantees that the user understands the exact "thought process" behind the translation.
