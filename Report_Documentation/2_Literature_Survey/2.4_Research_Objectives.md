# 2.4 Research Objectives

Based on the limitations identified, our project establishes the following core research objectives:

1. **Develop a Schema-Aware Pipeline**: To construct an NLP engine that semantically tags natural language tokens and cross-references them against an established database schema to prevent hallucinated columns or tables.
2. **Implement Graph-Based Join Inference**: To build an algorithm (such as Breadth-First Search) capable of automatically inferring and mapping foreign key relationships between disparate tables mentioned in a user's query.
3. **Enhance User Trust via Visualization**: To create a transparent graphical user interface that visually demonstrates the intermediate states of the translation pipeline, allowing non-technical users to verify the logical mapping of their inputs before execution.
4. **Achieve Reliable Execution**: To ensure that 100% of the successfully generated queries are syntactically valid and structurally aligned with the target database.
