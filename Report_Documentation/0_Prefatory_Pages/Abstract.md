# Abstract

In today's data-driven world, relational databases store the vast majority of critical information. However, querying these databases requires proficiency in Structured Query Language (SQL), which acts as a massive barrier for non-technical professionals. To address this, Natural Language to SQL (NL2SQL) systems have emerged, aiming to allow users to ask questions in plain English. Recent approaches rely heavily on Large Language Models (LLMs); however, these "black box" models are notoriously prone to hallucinations, often generating syntactically valid but structurally incorrect queries that can wreak havoc in enterprise environments.

This project introduces a robust, schema-aware, rule-guided approach to NL2SQL translation. By steering away from purely probabilistic generation, our hybrid system grounds the natural language processing (NLP) pipeline strictly within the actual database schema. The system tokenizes input, applies semantic tagging, and utilizes a graph-based inference algorithm to trace foreign-key relationships. This ensures absolute adherence to the database architecture. Furthermore, to combat the lack of user trust inherent in AI systems, we developed an interactive frontend visualization. This interface breaks down the internal "thought process" of the NLP engine, allowing users to visually verify how their words map to tables and columns before any query is executed. The result is a highly transparent, hallucination-free querying tool tailored for business environments.
